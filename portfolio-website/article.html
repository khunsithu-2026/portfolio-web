Introduction to CSS Grid and Flexbox: A Modern Layout Guide

Khun Si Thu
• October 12, 2026 • 5 min read

Modern responsive layouts lean on two core tools: Flexbox and CSS Grid. Used together, they cover almost every interface pattern you’ll need.

One dimension: Flexbox

Flexbox lays out items along a single axis—row or column. It shines when distributing space, aligning items, and building components like navbars or form rows.

For example, a header with a logo on the left and links on the right is a natural Flexbox problem: justify-content: space-between; replaces fragile float hacks.

Flexbox is ideal inside components. When you need to line up a control next to a label, reach for Flexbox first.

Two dimensions: Grid

Grid defines rows and columns at once. Use it for page-level structure, galleries, and dashboards where placement on both axes matters.

Explicit tracks and line-based placement make complex layouts predictable without nesting many flex containers.

When to use which

  • Flexbox: Single-axis layouts—toolbars, chip rows, centered overlays.
  • Grid: Macro layouts spanning rows and columns—cards, bento-style sections, editorial pages.

In practice, combine them: Grid for the skeleton, Flexbox inside each region for alignment and wrapping behavior.
